# Quick Notes – Chrome Extension

A lightweight Chrome extension that lets you **take quick notes from any website** and automatically **saves the current page URL** with your note. You can also **download your notes as text files** and clear them anytime — all within a simple popup.

---

## Features
- Write and save quick notes instantly  
- Automatically include the current website URL  
- Download notes as `.txt` files  
- Notes are stored locally using Chrome Storage  
- One-click clear option  

---

## Built With
- **JavaScript (Vanilla JS)**  
- **Chrome Extension APIs** – `storage`, `tabs`, `downloads`  
- **HTML + CSS** for a clean popup UI
